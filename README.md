# Prospect Sites

Professional website template for plumbing businesses. Designed for quick deployment and customization via environment variables.

## Customization

Set these environment variables to customize for each business:

```
NEXT_PUBLIC_BUSINESS_NAME=Business Name Here
NEXT_PUBLIC_BUSINESS_PHONE=Phone Number
NEXT_PUBLIC_BUSINESS_LOCATION=City, State
NEXT_PUBLIC_BUSINESS_ADDRESS=Full Address
NEXT_PUBLIC_BUSINESS_RATING=5.0
NEXT_PUBLIC_BUSINESS_REVIEWS=10
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import to Vercel
3. Set environment variables in Vercel dashboard
4. Deploy

### Local Development

```bash
npm install
npm run dev
```
